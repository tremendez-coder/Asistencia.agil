El problema actual

Errores Humanos

Errores humanos frecuentes en la asistencia manual afectan la precisión de los datos.

Tiempo Perdido

Minutos valiosos perdidos al inicio de cada clase reducen el tiempo efectivo de enseñanza.

Datos Poco Confiables

Datos poco confiables que dificultan la gestión educativa efectiva.

Características principales

Reconocimiento Facial

Tecnología avanzada con OpenCV y face_recognition para identificación precisa.

  • Precisión de más del 90% en condiciones óptimas
  • Detección en tiempo real
  • Funciona con webcams estándar

Reportes Inteligentes

Generación automática de reportes y exportación en Excel desde base de datos SQLite.

  • Exportación automática por curso y fecha
  • Estadísticas de asistencia en tiempo real

Gestión y Control de Asistencia

Panel de gestión para registrar, justificar y auditar asistencias con trazabilidad.

  • Marcación automática y manual con validaciones
  • Justificaciones con evidencia y historial
  • Filtros por curso, fecha y estudiante
  • Exportación y auditoría de cambios

Nuestra solución

El sistema captura rostros con una webcam estándar (1080p), procesa con IA en Python y registra la asistencia en una base de datos SQLite. Una plataforma web en Flask permite a administradores y preceptores visualizar, justificar y exportar asistencias.
